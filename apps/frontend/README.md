# Birthday Frontend

Minimalist UI for managing birthday notifications.

## Stack
- Vite + Vue 3 (TypeScript)
- TailwindCSS v4
- Cloudflare Pages / Generic SPA

## Setup
1. `pnpm install`
2. `pnpm dev`
3. Ensure Backend is running on port 8787 (`cd ../backend && pnpm dev`)

## Features
- Create User (Name, Email, Birthdate, Timezone)
- List Users
- Delete User
